# Calculator using C 🧮

Upgrading my calculator game from Python to C 🚀! After building a calculator in Python, I wanted to jump into a lower-level language to understand how pure programming logic works without the easy syntax shortcuts.

---

## 🚀 Features
* **Core Arithmetic Operations:** Supports Addition, Multiplication, Division, and Subtraction.
* **User-Friendly Menu:** Clear prompt-based menu for choosing operations.
* **Pure C Logic:** Written completely from scratch in C, using standard input/output (`stdio.h`).

---

## 🛠️ How It Works
The program prompts the user to enter two numbers ($a$ and $b$), displays an operation menu, and processes the user's choice:

1. **Sum (+)**
2. **Multiplication (*)**
3. **Division (/)**
4. **Subtraction (-)**

---

## 💻 Tech Stack
* **Language:** C
* **Tools:** VS Code, GCC Compiler

---

## 📖 What I Learned
* Moving away from Python's easy features (like `elif`) to understand standard C switch/if-else architecture.
* Handling formatted inputs and outputs using `scanf()` and `printf()`.
